# 《拯救古之岚》游戏设计文档

## 1. 游戏概述
- **游戏名称**：拯救古之岚
- **核心玩法**：幸存者类（Survivor-like）/ 肉鸽（Roguelike）。参考《土豆兄弟》（Brotato），玩家在限定时间内抵御波次进攻。
- **题材**：剑网三（JX3）武侠背景。
- **胜利条件**：成功挑战 20 波敌人。

## 2. 角色系统
- **初始角色**：祁进（纯阳）。
- **可解锁角色**：
  - **李承恩（天策）**：高血量、高防御。
  - **更多角色**：后续扩展万花、藏剑等经典门派角色。
- **角色差异**：
  - 不同的初始属性（攻击力、攻速、移速、血量、防御）。
  - 武器倾向（如纯阳倾向气劲，天策倾向长兵）。
  - 独特的被动技能。

## 3. 战斗机制
- **视角与布局**：2.5D/3D 横版固定区域（使用 R3F 渲染）。
- **位置**：我方位于屏幕左侧，敌人从右侧不断涌入。
- **波次时间**：每波战斗坚持 30 秒即视为通关。
- **难度曲线**：敌人数量、血量、伤害随波次线性/指数增长。
- **同伴系统**：玩家可以在商店招募 NPC 士兵（如天策府将士、纯阳弟子）协助防守，增加防线的厚度。

## 4. 经济与成长系统
- **货币获取**：击杀敌人掉落、波次结算奖励。
- **备战阶段（商店）**：
  - **属性增强**：购买卡片提升暴击、吸血、闪避等属性。
  - **武器与装备**：购买并升级门派特色武器。
  - **招兵买马**：招募并升级场上的协助 NPC。
  - **刷新商店**：消耗金钱随机化商品。
