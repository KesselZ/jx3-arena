# 剑三竞技场 (JX3-Arena) 属性系统设计方案

本文档定义了游戏中的核心属性体系，参考 Roguelike 游戏（如《土豆兄弟》）的数值架构，为后续的增益（Buff）系统和装备系统提供基础。

## 1. 基础输出属性 (Base Offense)
这些属性直接决定单位的杀伤效率。

| 属性名称 | 描述 | 核心作用 |
| :--- | :--- | :--- |
| **伤害 (%)** | 最终伤害加成系数 | 提升所有攻击方式的最终输出数值。 |
| **攻击速度** | 武器出手频率 | 缩短攻击间隔，提升单位时间内的输出。 |
| **攻击范围** | 攻击覆盖空间 | 增加弹道飞行距离、近战挥砍半径或爆炸覆盖面。 |
| **暴击率** | 触发倍率伤害的概率 | 增加输出爆发性，最高通常上限为 100%。 |
| **暴击伤害** | 暴击时的伤害倍率 | 决定触发暴击时的伤害强度（基础通常为 200%）。 |

## 2. 机制增强属性 (Mechanics)
影响攻击的特殊物理表现和控制效果。

| 属性名称 | 描述 | 核心作用 |
| :--- | :--- | :--- |
| **穿透 (Pierce)** | 弹道贯穿敌人的次数 | 远程武器（如飞剑、暗器）的核心对群属性。 |
| **击退 (Knockback)** | 受击者的位移距离 | 增强防守拉扯能力，防止被近战敌人包围。 |

## 3. 元素伤害属性 (Elemental)
具有特殊逻辑的额外伤害类型。

| 属性名称 | 描述 | 核心作用 |
| :--- | :--- | :--- |
| **毒性 (Poison)** | 持续伤害 (DoT) | 基于击中伤害百分比，使目标在一段时间内持续掉血。 |
| **阴性 (Yin)** | 额外伤害 + 减速 | 附带固定额外伤害，并降低目标的移动速度。 |
| **阳性 (Yang)** | 易伤效果 | 为目标施加 Debuff，提升其后续受到的所有伤害。 |

## 4. 生存与功能属性 (Defense & Utility)
决定玩家的生存能力和发育效率。

| 属性名称 | 描述 | 核心作用 |
| :--- | :--- | :--- |
| **最大生命值** | 基础容错空间 | 提升角色能承受伤害的总量。 |
| **生命恢复** | 自动回血能力 | 每秒固定回复一定数值的生命值。 |
| **生命偷取** | 攻击吸血 | 通过攻击频率或概率，将伤害的一部分转化为自身生命。 |
| **衰减型护甲** | 非线性减伤 | 公式：$100 / (100 + Armor)$。护甲越高，边际效益递减。 |
| **闪避** | 概率免疫伤害 | 概率完全回避单次伤害（上限通常设为 60%-70%）。 |
| **移动速度** | 位移效率 | 影响走位拉扯效率，躲避技能的关键。 |
| **拾取范围** | 自动吸附半径 | 增加经验球和掉落物的吸附距离，优化发育体验。 |

---

## 5. 数值公式参考
*   **最终伤害** = `基础伤害 * (1 + 伤害%) * (是否暴击 ? 暴击伤害 : 1) * (1 - 护甲减伤率)`
*   **护甲减伤率** = `护甲值 / (100 + 护甲值)`
*   **攻击间隔** = `1 / (基础攻速 * (1 + 攻速加成%))`
